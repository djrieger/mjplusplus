void firm::visitor::ClassVisitor::visit ( shptr< const ast::ClassDeclaration >  classDeclaration)
virtual

Visits a ClassDeclaration. Sets its owner to the the class type that's retrieved from the declaration's ident with the help of FirmInterface. Then uses a MemberVisitor to visit the declaration's members.
